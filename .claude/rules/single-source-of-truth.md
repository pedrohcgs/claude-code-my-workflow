# Single Source of Truth: Enforcement Protocol

**The LaTeX chapter files (`Chapters/*.tex`) are the authoritative source for ALL written content.** Figures and tables are derived from scripts.

## The SSOT Chain

```
Chapters/*.tex (SOURCE OF TRUTH — written content)
  │
  ├── Figures/  ← generated by scripts/python/ or scripts/R/
  │               NEVER edited manually
  │
  ├── output/tables/  ← generated by scripts/R/
  │                     NEVER edited manually
  │
  ├── output/models/  ← generated by scripts/python/
  │
  └── Bibliography_base.bib  (shared, curated manually)

NEVER modify a figure or table directly.
ALWAYS regenerate from the source script.
```

---

## Figure Provenance Protocol (MANDATORY)

**Every figure in `Figures/` must have a corresponding script.**

### Before using or referencing any figure:

1. Identify the generating script — it must exist in `scripts/python/` or `scripts/R/`
2. The script must have a comment at the top listing its outputs:
   ```python
   # OUTPUTS: Figures/maps/ndvi_trend_sidama.png
   ```
3. If no generating script exists, the figure is **untracked** — flag this and create the script before committing

### When a figure needs to be updated:

1. Modify the generating script
2. Re-run the script to regenerate the figure
3. Verify the new figure visually before committing
4. Never overwrite `Figures/` manually

---

## Table Provenance Protocol (MANDATORY)

**Every regression table, summary statistics table, and model output in the thesis must trace to an R script.**

1. R scripts producing tables go in `scripts/R/`
2. Output `.tex` table files go in `output/tables/`
3. Tables are `\input{}`'d into LaTeX chapters — never copy-pasted manually
4. Script must log the exact command used: `stargazer(...)`, `kableExtra(...)`, `modelsummary(...)`

---

## Content Fidelity Checklist (Chapter Submission)

```
[ ] All figures cited in chapter exist in Figures/ with a generating script
[ ] All tables cited in chapter exist in output/tables/ with a generating R script
[ ] All \cite{} keys exist in Bibliography_base.bib
[ ] No figure or table was edited manually (verify via git diff)
[ ] Notation in chapter matches knowledge-base-template.md registry
[ ] Variable names in text match variable names in scripts exactly
```

---

## What This Rule Does NOT Cover

- Beamer/Quarto slide sync (not applicable to this thesis)
- TikZ diagram freshness (not applicable; use Python-generated figures instead)
