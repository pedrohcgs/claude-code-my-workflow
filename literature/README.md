Organized paper notes and synthesis tables for the Nepal & Global Energy Economics project.

---

## Structure

```
literature/
├── README.md          # This file — protocol for adding paper summaries
├── global/            # Global energy economics papers (2023–2026 focus)
└── nepal/             # Nepal-specific energy papers
```

---

## Protocol for Adding Paper Summaries

### File naming
`AUTHORLASTNAME_YEAR_keywords.md` — e.g., `apergis_2024_energy_growth_panel.md`

### Required fields (use template below)

```markdown
# [Full Paper Title]

**Authors:** Last, F.; Last, F.
**Year:** YYYY
**Journal:** [Journal name]
**DOI / URL:** [link]
**Date added:** YYYY-MM-DD

---

## Research Question
[One sentence]

## Data
- Source(s): [IEA, WDI, NEA, etc.]
- Coverage: [countries / years]
- Key variables: [list]

## Method
[Estimator, identification strategy, key assumptions]

## Main Findings
- [Finding 1]
- [Finding 2]

## Contribution / Gap Addressed
[Why this paper matters; what it adds to the literature]

## Limitations / Critiques
[What the paper acknowledges; what you observe]

## Relevance to Our Project
- [ ] Global context
- [ ] Nepal context
- [ ] Methodological inspiration
- [ ] Proposal seed

## Citation (BibTeX key)
`[bibtex_key]` — add full entry to `Bibliography_base.bib`
```
