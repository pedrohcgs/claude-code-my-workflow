\begin{abstract}
\noindent
本文档详细阐述了四个核心异质性变量的测度方法论：
（1）地方保护主义指数 $Protection_d$——基于双向固定效应面板残差法与极差标准化；
（2）企业外生技术扩张阻力指数 $Friction_{i,t,c}$——融合企业技术组合、宏观技术互补规律与空间制度壁垒的三步构建法；
（3）异质性分组设定——基于处理组事前均值切分；
（4）央地协同（跨区资本网络）指标——深度与广度双维度测度。
附录以"星河造车"虚构案例进行沙盘推演，完整演示 $Friction$ 指数的数值计算过程。
\end{abstract}
