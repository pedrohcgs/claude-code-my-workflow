Course Website

GitHub Pages Deployment Target

This directory (docs/) is where rendered Quarto slides are deployed for GitHub Pages.

Run ./scripts/sync_to_docs.sh to render and deploy your slides here.

Getting Started

  1. Create your first lecture in Quarto/ (e.g., Lecture1_Topic.qmd)
  2. Run ./scripts/sync_to_docs.sh Lecture1 to render and deploy
  3. Enable GitHub Pages in your repo settings (source: docs/ folder)
  4. Replace this placeholder with your course homepage

Slides

Rendered slides will appear in docs/slides/ after running the sync script.
