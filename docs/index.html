Claude Code Academic Workflow

Work in progress. This is a living document that I keep updating as I learn new things. It's mostly a summary of how I've been using Claude Code for academic work — shared here so friends and colleagues can take a look.

A ready-to-fork starter kit for academics using Claude Code with LaTeX/Beamer, R, and Quarto. Contains a complete multi-agent infrastructure — specialized reviewers, quality gates, adversarial QA loops, and automated workflows — extracted from a production PhD course.
